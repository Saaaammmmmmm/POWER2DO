# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/515c7819-22a9-4ef9-af90-2b1a54eadd19

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy for Free on GitHub Pages

This app is now configured to publish as a free GitHub Pages site.

1. Push these changes to GitHub.
2. In your repository settings, open Pages and choose "GitHub Actions" as the source.
3. GitHub will build and publish the site automatically from the workflow in [.github/workflows/deploy.yml](.github/workflows/deploy.yml).
4. After the workflow finishes, your site will be available at:
   `https://Saaaammmmmmm.github.io/POWER2DO/`
